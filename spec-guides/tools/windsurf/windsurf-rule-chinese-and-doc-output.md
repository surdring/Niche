# Windsurf Workspace Rule：中文回复 & 文档落盘

> 建议在 Windsurf 的 `Customizations → Rules → + Workspace` 创建一条规则，粘贴本文件内容，并将 Activation Mode 设为 **Always On**。

<rules>

## 1) 语言
- 始终使用中文回复（除非用户明确要求使用其他语言）。

## 2) 文档生成与落盘
- 当用户要求“生成文档/规范/模板/清单”等内容时：
  - 必须根据仓库目录结构，选择**合理的目标目录**。
  - 必须以 **Markdown（.md）** 格式在目标目录中**创建文件并写入内容**（而不是只在聊天中输出）。
  - 若用户未指定目标目录或命名：
    - 先提出 1-3 个建议路径与文件名供用户确认，再创建文件。

</rules>
